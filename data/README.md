# Data

## data/sample/

`oxide_interface_d1000.npz` is a committed synthetic sample: one simulated
three-phase spectrum image (64 x 64 pixels x 200 channels, 380 to 778 eV,
dose 1000 counts per pixel spectrum, 1.5 channels of energy drift) produced
by this repository's own simulator with a fixed seed:

    eelsunmix simulate --dose 1000 --seed 0 --output data/sample/oxide_interface_d1000.npz

It contains the noisy counts cube (uint16), the energy axis, and the exact
ground truth (endmember spectra, abundance maps, drift field), so the demo
and the tutorial notebook run and score with no download. All of it is
synthetic and generated by committed code; there is no external data source
and no license beyond this repository's MIT license.

## Real data

No real spectrum image is committed: experimental EELS/EDX maps with a
clearly stated open license are rare, and this repository does not ship
anything with unclear provenance. To run on your own data, see
`examples/unmix_your_own_map.py` and `docs/api.md` (section "Bring your own
data"): save your cube as `.npy` or `.npz` shaped `(ny, nx, n_channels)`
with non-negative counts, and any of the methods will decompose it. Ground
truth scoring is only available for simulated scenes.
